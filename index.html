Problem: Convert Celsius to Fahrenheit The algorithm to convert from Celsius to Fahrenheit is the temperature in Celsius times 9/5, plus 32. You are given a variable celsius representing a temperature in Celsius. Use the variable fahrenheit already defined and assign it the Fahrenheit temperature equivalent to the given Celsius temperature. Use the algorithm mentioned above to help convert the Celsius temperature to Fahrenheit.

Expected Outcome: Given a certain argument representing temperature in Celsius, function calculates fahrenheit.

Solution:


    function convertToF(celsius) {
    let fahrenheit= (celsius * 9/5) + 32;
    return fahrenheit;
    }

convertToF(30);

Interview Answer

In order to calculate Farenheit from an argument representing Celsius, we must first know the algorithim required to calculate this desired result. We are given this algorithim in the question: "Temperature in Celsius times 9/5, plus 32".

Next, we determine what we need to instruct the computer to do. The computer must utilize 'order of operations' which is BODMAS (Brackets, Orders, Division, Multiplication, Addition, Subtraction) to calculate our formula.

Therefore, our syntax becomes (open parenthesis 9 divided by 5 times 'Celsius' close parenthesis plus 32?)

We pass an argument into our function-call; converToF(). This takes the place of our function parameter (celsius) and is used in the calculation of our function expression.
